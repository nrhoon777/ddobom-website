# 또봄 (한국청년암협회) 홈페이지 리뉴얼

사단법인 한국청년암협회 **또봄**(Korea Young Cancer Association)의 홈페이지 리뉴얼 프로젝트입니다.
국내 최초 20·30대 청년 암 환자·경험자 지원 비영리단체로, '또봄'은 '다시 봄'(희망)을 의미합니다.

> 기존 사이트가 모바일 전용으로만 최적화되어 있어, **데스크톱·태블릿·모바일 모두 대응하는 반응형**으로 전면 개편합니다.

## 현재 상태 — 1차 초안 (정적 사이트)

기획서(`KYCA_홈페이지_리뉴얼_기획서.md`)를 기준으로 제작한 반응형 정적 초안입니다.

| 파일 | 설명 |
|---|---|
| `index.html` | 홈 (히어로·미션·임팩트·사업활동·후원 CTA·소식·참여) |
| `about.html` | 또봄 소개 |
| `programs.html` | 사업·활동 (또봄클래스·앎 캠페인·Cheer UP 곧·봄·위케어리셋·또또봄TV) |
| `donate.html` | 후원하기 (정기/일시·금액 선택·기부금영수증·투명성) |
| `assets/css/style.css` | 디자인 시스템(봄 팔레트)·반응형·접근성 |
| `assets/js/main.js` | 모바일 메뉴·후원 금액 선택 |

### 적용 원칙
- 모바일 우선 반응형, 후원 상시 CTA
- '다시 봄' 희망 톤 (투병 클리셰·공포 이미지 배제, 환자 신원 비식별)
- KWCAG 웹접근성 기초 (스킵링크·시맨틱·포커스·44px 터치·대체텍스트)

### 초안 한계 (교체 예정)
실제 로고/BI·사진·통계·법인/계좌 정보는 예시(placeholder)이며, 결제(PG)·백엔드·CMS·회원/커뮤니티는 미연동(데모) 상태입니다.

## 로컬에서 보기
별도 빌드 없이 `index.html`을 브라우저로 열면 됩니다.

```bash
# (선택) 간단한 로컬 서버
python -m http.server 8000
# http://localhost:8000
```

## 기술 스택 (초안)
순수 HTML / CSS / JavaScript (정적). 향후 기획서에 따라 Next.js + 헤드리스 CMS + 후원 결제 백엔드로 확장 예정.
